# Descrizione greet.php

## Scopo dello script:

Lo script deve restiruire una stringa in cui viene salutata la stringa che immetti 


## Prerequisiti:

PHP


## Comandi di esecuzione con esempi:

1. php greet.php
2. php greet.php flori
3. php greet.php J.B.


## Output atteso:

1. Ciao, Mondo!
2. Ciao, flori!
3. Ciao, J.B.!
